## 项目概述

本项目聚焦于“**研学旅游舆论数据的获取与空间分析**”，通过网络爬虫与自然语言处理技术，对研学景区评论数据进行采集、清洗与分析。项目目标包括：

- 构建覆盖全国多个省份与景区的研学舆论数据集；  
- 识别研学旅游的**时空热点区域**与**评论情感特征**；  
- 通过语义网络与主题分析，揭示不同地区研学产品的口碑差异；  
- 提供可复现的分析流程，支持后续科研或政策研究使用。

## 模块说明


### 1. 数据获取模块（0_）
功能说明：
- 负责爬取舆论评论数据，按省份与景区分类存储
- 包含爬取进度监控与异常重试机制
- 输出统一格式的 df_all.csv 数据文件，便于后续分析

### 2. 研学热点区域分析（3_）
功能说明：
- 识别不同时间段与城市间的研学热点
- 量化各类型产品的时空分布变化
- 绘制多阶段的研学活动趋势与空间聚集图

主要功能包括：
- 各类型与城市的研学产品数量、比例计算
- 不同年龄段研学游的时间趋势折线图
- 冷热点识别分析（基于 libpysal.weights）
- 核密度分析与 DBSCAN 聚类识别
- 各阶段热点景点类型对比与时序可视化

### 3. 舆情数据分析（4_）
分析目标：
- 研究研学产品在舆论中的情感分布与语义特征
- 提取高频主题与地区差异
- 可视化舆论网络结构与主要话题

包含以下部分：
- **舆论描述性分析**  
  - 各类型研学景点好评率箱型图  
  - 好评率地理分布图  

- **语义网络分析**  
  - 好评与差评词云  
  - 基于 `networkx` 的语义共现网络  

- **区域舆论分析**  
  - 各地区评论数量与情感比例条形图  
  - 不同省份的 LDA 关键词主题提取结果  

- **专题研究**  
  - 评论关键词词云  
  - 不同地区主题分布表格与特征比较

## 数据说明

| 数据文件 | 内容说明 |
|-----------|-----------|
| `data/comments_result_补充/` | 原始爬虫数据，按省份与景区分类 |
| `data/df_all.csv` | 合并后的全部评论数据（约 100 万条） |
| `data/省份景区概况.csv` | 各省份研学产品的平均分、好评率、评论数与 POI 数量统计 |

数据来源自在线旅游平台与研学相关公开数据接口，仅用于学术分析与模型验证。

## ▶运行示例

以下为项目运行的推荐步骤（以 VSCode 为例）：

```bash
# 1. 创建虚拟环境
conda create -n edu_tourism python=3.11
conda activate edu_tourism

# 2. 安装依赖
pip install -r requirements.txt  # 或手动安装上述依赖包

# 3. 打开 VSCode 或 JupyterLab
code .   # 打开项目文件夹

# 4. 运行 Notebook
# 按顺序执行 0_*.ipynb → 3_*.ipynb → 4_*.ipynb
```

运行后将在 `/data/` 目录中生成分析结果文件与中间输出数据。

## 方法与技术栈

| 模块 | 方法说明 | 主要依赖 |
|------|-----------|----------|
| 数据爬取 | 模拟请求、JSON 解析、数据清洗 | `requests`, `json`, `pandas` |
| 空间分析 | 冷热点识别、核密度估计、DBSCAN 聚类 | `libpysal`, `seaborn`, `scikit-learn` |
| 舆论分析 | 情感分析、词频统计、LDA 主题建模 | `snownlp`, `gensim`, `matplotlib` |
| 语义网络 | 共现矩阵构建与可视化 | `networkx`, `wordcloud` |
